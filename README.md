# Smart Vision Assistant
### AI-Powered Wearable Vision Companion for Visually Impaired Individuals


---

## Overview

Smart Vision Assistant V4U2C is an AI-powered wearable assistive system designed to improve environmental awareness for visually impaired individuals. The system combines real time object detection, optical character recognition (OCR), and speech synthesis to provide instant audio feedback using a Raspberry Pi and camera module.

The application continuously captures images from the camera, detects surrounding objects using the YOLOv8 deep learning model, recognizes printed text using Tesseract OCR, estimates the relative position of detected objects, and converts the extracted information into speech.

The project demonstrates how embedded AI and computer vision can be integrated into a portable, low cost assistive device.

---

## Features

- Real time object detection using YOLOv8
- Optical Character Recognition (OCR)
- Text-to-Speech (TTS) feedback
- Left, Front, and Right object localization
- Approximate object distance estimation
- Raspberry Pi compatible
- Portable wearable implementation
